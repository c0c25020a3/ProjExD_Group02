# こうかとんが転んだ

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* こうかとんがこちらを見てくるタイミングで動かないようにして、無効を向いている間にこうかとんのところまでたどり着こう！

## ゲームの遊び方
* 矢印キーでプレイヤーを操作して、こうかとんのいるところまでたどりつけばクリア
* こうかとんがこちらを向いているときに動いてしまうと、ゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 左から右に向かう
* タッチしたらクリア
* 振り向いているときに動いたらゲームオーバー
* 不定期で振り向く
* リスタート機能
* 音声
* 足音

### 分担追加機能
* マップ上に遮蔽を設置
* 時間停止&経過時間の表示（担当：シュエー）
    * スペースキー押下で発動する特殊スキル
    * 発動中はこうかとんの動きが完全に停止する
    停止時間：3秒
    クールタイム：5秒
    * 画面上にクールタイム残り時間を表示
    * ゲーム開始からの経過時間を画面左上に表示
    * 秒単位でカウントアップ
    * クリア・ゲームオーバー時にも反映される
* ライフ
* クリア後次のステージにいって難易度が上がる(担当：宮川)　：1回以上クリアすると、15秒ごとに爆弾が複数生成される。クリアするごとに増加
* スタミナ（担当：本永）：SHIFT+十字キーでダッシュができる。スタミナ制限付き
